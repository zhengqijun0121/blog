  1. 登录和退出

从某些私有仓库拉取 Docker 镜像，可能需要进行登录。

登录命令如下：

1

docker login url

登陆成功后，密码会被加密保存在 ~/.docker/config.json 文件中。

退出命令如下：

1

docker logout
  1. 拉取镜像
1

docker pull image_name:tag

如果没有指定 tag，默认为 latest。

  1. 推送镜像
1

docker push username/image_name:tag
